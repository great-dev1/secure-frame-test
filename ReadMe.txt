To set-up frontend execute below steps to run the project root:
- npm install
- npm run dev (it will start the frontend)

To set-up backend execute below steps to run the project root:
- npm install
- npm start (to start backend server)
- call - POST {base_url}/api/add-data (it will read data from CSV and insert it into database MongoDB)

Note: You need to setuo MongoDB database first.
